# 分镜脚本 API 接口文档

## 概述

本技能根据产品白底图 + 商品卖点JSON + 视频类型 + 品类 + 国家 + 时长，自动匹配爆款镜头结构生成10s/15s分镜脚本。输出JSON供Skill3消费 + Markdown供人工审阅。

## 输入

```json
{
  "product_white_img": "E:\\path\\product_layer.png",
  "product_name": "商品全称",
  "商品名称": "商品全称",
  "核心卖点": {
    "主卖点": "核心优势",
    "次卖点": ["次卖点1", "次卖点2", "..."]
  },
  "适用人群": "目标人群",
  "适用场景": "使用场景",
  "category": "家居日用|美妆护肤|食品零食|服饰鞋包|数码3C",
  "video_type": "痛点解决|UGC种草|产品演示|开箱种草|CTA带货",
  "hook_points": "吸睛点（可空）",
  "target_duration": 10,
  "price": "价格（可空）",
  "target_market": "thailand|cn|jp|kr|us"
}
```

## 输出JSON结构（3段式）

```json
{
  "{产品名称}产品参考图": "@产品白底图路径",
  "人物形象特征参考": "人物外貌/妆容/穿搭文本",
  "镜头脚本": [
    {
      "镜号": 1,
      "时长": "3s",
      "景别": "特写",
      "运镜": "固定",
      "角度": "平视",
      "画面": "画面详细描述",
      "本地台词": "本地语言台词",
      "字幕": "字幕/弹幕文本",
      "音效/BGM": "音效+BGM描述",
      "光影": "光影色调",
      "真实约束": "真实感约束"
    }
  ]
}
```

## 核心规则

1. 仅10s/15s两种时长
2. 5种视频类型各有独立固定镜头结构
3. 5种品类各有独立镜头/音效/光影规则
4. 国家仅约束：人物特征、台词语种、场景辅助修饰
5. 全视频仅`@product_ref`一个视觉引用
6. 人物/背景纯文本生成，无额外图片输入
