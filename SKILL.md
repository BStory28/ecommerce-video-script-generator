# 国际化电商视频生成 — Skill2：视频脚本的生成

## 管线定位

```
Skill1 基图生成器 → product_layer.png + selling_points.json（含product_function + user_pain_point）
                    ↓
Skill2 分镜脚本生成器 ← 本技能
  输入: 产品白底图路径 + 商品卖点 + 视频类型 + 品类 + 功能属性 + 用户痛点 + 国家 + 时长
  输出: storyboard.json (AI管线) + storyboard.md (人工审阅)
                    ↓
Skill3 AI视频渲染 → 最终视频
```

## 核心原则

1. **全视频仅一张产品白底图**作为参考，无其他产品素材
2. **国家仅管控**：人物特征、台词/字幕语种、场景风格辅助修饰
3. **仅支持时长**：10秒 / 15秒
4. **人物、背景无图片输入**，全部文本生成
5. **功能属性规则优先级高于品类通用规则**，实现同一品类内差异化

## 视频类型固定镜头结构

### 1. 痛点解决（通用投流爆款）
镜头: 痛点冲击 → 产品亮相 → 实操演示 → 效果对比 → [15s增加]推荐CTA
时长: 10s=2/2/3/3s  15s=3/2/3/4/3s

### 2. UGC种草（生活化博主风）
镜头: 日常代入 → 产品全景 → 上手体验 → 亮点总结 → [15s增加]种草推荐
时长: 10s=2/2/3/3s  15s=3/2/3/4/3s

### 3. 产品演示（功能向）
镜头: 产品全景特写 → 功能演示1(主卖点) → 功能演示2(次卖点) → 整体效果展示
时长: 10s=2/2/3/3s  15s=3/2/3/4/3s

### 4. 开箱种草（新品类）
镜头: 开箱动作 → 产品完整露出 → 初次试用 → 细节亮点 → [15s增加]满意推荐
时长: 10s=2/2/3/3s  15s=3/2/3/4/3s

### 5. CTA带货（短平快转化）
镜头: 产品高光特写 → 核心卖点 → 快速实操 → 价格+下单引导
时长: 10s=2/2/3/3s  15s=3/2/3/4/3s

## 执行步骤

### Step1 输入解析
- 读取 `product_white_img`，绑定图片路径
- 解析 `target_market` → 确定人物特征、语种、场景修饰
- 解析 `video_type` → 调取对应镜头结构
- 解析 `category` → 匹配品类基础规则
- **解析 `product_function` → 匹配功能属性专属规则，覆盖品类通用规则中的冲突项**
- **解析 `user_pain_point` → 调整镜头重点（痛点解决类强化对比，UGC种草类强化真实感等）**
- 解析卖点 → 转化为可视化画面动作，按功能属性分配至镜头

### Step2 时长分配
按视频类型 + 时长(10/15s) + 镜头数自动分配，严格遵循时长表

### Step3 生成基础模块
- 项目基础信息
- 人物设定文本（仅由国家生成）
- 产品视觉基准说明
- 功能属性规则说明
- 用户痛点镜头策略说明

### Step4 生成镜头表格（核心）
每镜强制规则：
- 出现产品时直接用产品名称，不使用 @product_ref
- 台词/字幕使用目标语种
- 光影/音效/特写优先匹配功能属性规则，其次匹配品类规则
- 场景由视频类型+品类决定，仅轻叠加国家修饰

### Step5 输出双文件
- `storyboard.json` — AI管线机器读取（含产品功能属性+用户痛点元数据）
- `storyboard.md` — 人工审阅表格

## AI驱动说明

### 输入 → AI生成链路（优化后）

```
用户输入参数
    ↓
脚本构造结构化Prompt：
  ├─ 商品信息（产品名称、卖点、适用人群、场景）
  ├─ 国家配置（人物特征、语种、场景修饰）
  ├─ 视频类型框架（镜头结构、时长分配）
  ├─ 品类基础规则（节奏、运镜、音效、光影基线）
  ├─ 功能属性规则（覆盖品类通用规则中的差异化项）
  ├─ 用户痛点指令（调整镜头重点和情感曲线）
  ├─ 细节说明（--details，可选，用户填写后严格参考到画面描述）
  ├─ 视频角度（--angle，可选，用户填写后严格参考到镜头结构编排）
  └─ 画面提示（产品白底图引用、真实约束）
    ↓
调用 gpt-5.4-mini → 生成JSON格式镜头脚本
    ↓
解析 + 格式化 → 输出JSON + Markdown
```
